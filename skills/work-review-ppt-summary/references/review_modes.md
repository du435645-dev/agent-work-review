# Review Modes

## `phase-review`

Default for periodic work recap and stage review.

- background
- output
- impact
- next_steps

Recommended guizang mode:
- swiss
- 10-14 slides

## `self-review`

Use when the user is preparing a personal self-review or promotion-defense draft.

Keep the same structural base, but highlight:
- ownership
- decisions
- reusable methods

## `formal-report`

Use when the user wants a more neutral work report instead of a personal narrative.

Keep the same base fields, but write in a less self-centered tone and emphasize:
- scope
- delivery
- evidence
